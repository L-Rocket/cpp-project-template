# test

这是一个 C++ 项目模板，包含标准的文件夹结构：

- `src/`：源代码文件
- `include/`：头文件
- `tests/`：测试代码
- `third_party/`：第三方库
- `build/`：编译输出（自动生成）
- `CMakeLists.txt`：CMake 构建脚本

## 快速开始

1. 在 `src/` 里添加你的 C++ 源文件。
2. 在 `include/` 里添加头文件。
3. 用 CMake 构建项目。

```bash
mkdir build
cd build
cmake ..
make
```

4. 在 `tests/` 里添加测试代码。
